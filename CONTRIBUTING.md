# Contributing

This is a small, prompt-driven kit. Improvements welcome.

## Good contributions

- Tighter or clearer prompt wording (keep the no-fabricated-metrics rule intact)
- New prompt modules (e.g. recruiter cold-outreach, referral mapping, résumé pass)
- Variants for non-tech fields (the current examples lean tech/SRE/PM)
- Localization notes (e.g. how the method differs for EU / UK / India markets)

## Ground rules

- **Never bake in fake metrics or example numbers that could get copied verbatim.**
  Keep `[METRIC NEEDED]` placeholders.
- Keep prompts copy-pasteable: fenced code block, no smart quotes.
- One prompt = one job. Don't merge steps back into a mega-prompt.
- Preserve attribution to the original author in the README.

## How to propose a change

1. Fork the repo.
2. Make your edit in a branch.
3. Open a PR describing what problem the change solves.
